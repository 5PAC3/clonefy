Clonefy Music Platform

Descrizione:
Clonefy è una piattaforma di streaming musicale simile a Spotify. Gli utenti possono registrarsi, effettuare il login e ascoltare musica. Gli artisti possono caricare le proprie canzoni e gestire il loro contenuto.

Funzionalità
    Registrazione e login utenti
    Sistema di autenticazione sicuro
    Interfaccia utente moderna con tema scuro/viola
    Database per memorizzare utenti e canzoni

Tecnologie Utilizzate
    PHP
    MySQL
    HTML5/CSS3
    JavaScript
    Bootstrap 5

Requisiti di Sistema
    PHP 7.4 o superiore
    MySQL 5.7 o superiore
    Apache
    Browser web moderno

Clonefy Music Platform
Description

Clonefy is a music streaming platform similar to Spotify. Users can register, log in, and listen to music. Artists can upload their own songs and manage their content.

Features
    User registration and login
    Secure authentication system
    Modern user interface with dark/purple theme
    Database for storing users and songs

Technologies Used
    PHP
    MySQL
    HTML5/CSS3
    JavaScript
    Bootstrap 5

System Requirements
    PHP 7.4 or higher
    MySQL 5.7 or higher
    Apache
    Modern web browser
